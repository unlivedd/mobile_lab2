# Task List App

Android приложение для управления списком задач на Jetpack Compose.

## Возможности:
- ✅ Добавление задач
- ✅ Удаление задач
- ✅ Отметка выполненных задач
- ✅ Визуальное изменение выполненных задач
- ✅ Сохранение данных в Room Database
- ✅ Material Design 3

## Технологии:
- Kotlin
- Jetpack Compose
- Room Database
- MVVM архитектура
- Coroutines & Flow

## Запуск:
1. Откройте проект в Android Studio
2. Дождитесь синхронизации Gradle
3. Запустите на эмуляторе или устройстве
